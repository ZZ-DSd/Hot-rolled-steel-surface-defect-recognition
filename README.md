# Hot-rolled-steel-surface-defect-recognition
热轧钢生产过程中，表面缺陷（如划痕、氧化皮、裂纹、夹杂等）直接影响产品等级与客户验收。传统依赖人工目检方式存在效率低（≤2人/产线）、漏检率高（约15%~20%）、主观性强等问题。为提升质检自动化水平与产品质量一致性，主导构建一套端到端工业视觉缺陷检测系统，实现产线级实时识别与分类。
🛠️ 核心工作与技术实现
离线数据采集与质量保障：
协同产线工艺人员，在非生产时段/低速试运行阶段，使用工业相机（Basler acA2440-35uc）采集覆盖全工艺段（粗轧→精轧→卷取）的钢带表面图像共 11.7万张；
严格按缺陷类型、位置（边部/中部）、严重程度、光照/角度变化进行分层抽样，确保数据分布贴近实际产线工况；
建立三级质检机制：① 初筛（去模糊/过曝图）→ ② 工艺专家复核标注 → ③ 随机交叉校验（Kappa系数 ≥0.85），最终构建含6类典型缺陷（划痕、氧化铁皮、裂纹、折叠、夹杂、麻点）的高质量标注集，总计标注实例 42,300+ 个。
模型设计与鲁棒性增强：
采用 YOLOv8n-seg + Class-Aware Attention 架构，解决小目标（<32×32像素）与密集缺陷遮挡问题；
引入 Strong Augmentation Pipeline（包括模拟轧制抖动、金属反光噪声、局部遮挡Mask），显著提升模型对非理想成像条件的适应能力；
在跨批次测试中（不同日期/炉次采集），模型mailto:mAP@0.5达 93.1%，较未增强基线提升11.6%，证明其具备良好泛化性。
系统落地与价值闭环：
模型经TensorRT量化后部署于边缘服务器（Intel i7 + T4），支持单张图像批处理（≤15ms/图），满足后续对接在线系统的性能预留；
开发缺陷报告自动生成模块，输出含定位框、类别、置信度、建议处置等级的结构化JSON，供MES系统调用；
通过离线回溯验证：对3个月历史产线图像进行全量扫描，发现人工漏检缺陷147处，其中高风险缺陷（裂纹/折叠）占比38%，验证系统实用价值。
